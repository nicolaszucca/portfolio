I practice my knowledge in node, designing backend with security and authorization using express-session

Description

My main goal was to create a robust back-end API that was secure and required authentication to practice and hone my skills.
I used Nodejs with express for the server side. Express-validator for the middlewares and Express-session for the session data. I learned to use many libraries and concepts in this project, ejs for example. This library allowed rendering the views and using logic within them to receive the data and thus greatly facilitate the work.
